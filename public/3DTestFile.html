To generate your file in 3D
1. Place your file within the "Images" folder, remember the name of your file!
2. Within this repository there is a folder named "FloorplanToBlender3d", cd to this file within the terminal of your choice
3. Run the command "docker-compose run ftb"
4. Press enter to use default settings until the script asks "Do you want to set images to use in each config file?", enter 'y'
5. Enter the path to the file you would like to generate in 3D, this will likely be "Images/name_of_file.png"
6. Continue to press enter until the generation begins
7. Your file will be found in "FloorplanToBlender3d/Target/"
